max_heap:
    Условия:
1) Complete Binary Tree = все узлы должны быть заполнены слева направо. До этого не следующий этаж не переходим
2) значение в родителе `<=` значение в дочернем узле


min_heap:
    аналогично, только значение в родителе `>=` значение в дочернем узле

добавление/удаление/изменение за 0(logN), массив в кучу => O(n)
